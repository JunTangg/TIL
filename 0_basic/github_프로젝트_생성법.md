* 프로젝트 폴더 생성
* 기본 프로젝트 셋팅
* git으로 관리 시작

```sh
git init
git add .
git commit -m 'start-project'
```

* git 주소 설정

```sh
git remote add origin https://github.com/JunTangg/matcha.git
```

* 푸쉬

```sh
git push origin master
```
